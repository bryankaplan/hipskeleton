# hipskeleton

## Installation

    $ npm install
    $ npm run start

Local server: [http://localhost:3000/](http://localhost:3000/)

## Third-party hotel search API:

    https://www.hipmunk.com/interviews/hotel_search/scraper/<PROVIDER NAME>

Available provider names:

- "Expedia"
- "Orbitz"
- "Priceline"
- "Travelocity"
- "Hilton"
